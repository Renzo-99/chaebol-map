# DESIGN.md — ChaebolMap UI/UX 디자인 가이드

## 1. 디자인 컨셉

### 아이덴티티
- **전문적이고 신뢰감 있는** 금융/데이터 시각화 플랫폼
- **깔끔하고 미니멀한** 인터페이스로 복잡한 데이터를 명료하게 전달
- **다크 모드 기본** — 금융 터미널 느낌, 그래프 시각화에 최적

### 키워드
`Data-driven` · `Professional` · `Clean` · `Interactive` · `Trustworthy`

## 6. shadcn/ui 컴포넌트 활용 계획

### 필수 컴포넌트
| 컴포넌트 | 용도 |
|----------|------|
| `Button` | 모든 액션 버튼 |
| `Card` | 그룹 카드, 통계 카드, 회사 정보 카드 |
| `Dialog` | 모달 (회사 상세, 확인) |
| `Sheet` | 사이드 패널 (회사 상세 정보) |
| `Input` | 검색 입력 |
| `Select` | 그룹 선택, 필터 선택 |
| `Slider` | 지분율 임계값 필터 |
| `Badge` | 상장/비상장 태그, 업종 태그 |
| `Table` | 계열사 목록 테이블 |
| `Tabs` | 탭 네비게이션 (상세 정보 패널) |
| `Tooltip` | 노드/엣지 호버 시 정보 |
| `Command` | 통합 검색 (Cmd+K) |
| `NavigationMenu` | 상단 네비게이션 |
| `Separator` | 구분선 |
| `ScrollArea` | 스크롤 영역 |
| `Toggle` / `Switch` | 필터 토글 (상장사만 보기 등) |
| `Skeleton` | 로딩 상태 |
| `Breadcrumb` | 경로 표시 |
| `DropdownMenu` | 더보기 메뉴 |
| `Avatar` | 사용자 프로필 |
| `ThemeToggle` | 다크/라이트 모드 전환 |

## 9. 아이콘 시스템

- **아이콘 라이브러리**: Lucide React (shadcn/ui 기본)
- 주요 아이콘:
  - `Star` — 상장회사 표시
  - `Building2` — 회사
  - `Users` — 그룹/동일인
  - `GitBranch` — 지분 관계
  - `Search` — 검색
  - `Filter` — 필터
  - `Bookmark` — 북마크
  - `ZoomIn` / `ZoomOut` — 줌
  - `Maximize` — 전체 화면
  - `Download` — 내보내기
